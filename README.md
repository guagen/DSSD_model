# DSSD_model
这里面存了DSSD训练好的模型链接和相关的demo.py，对应mxnet版本的dssd模型（链接https://github.com/MTCloudVision/mxnet-dssd ）使用。
使用的时候把相关的文件给替换就可以跑demo了，该模型是我自己训练的一个dssd模型，321的图片大小，用的resnet101，在voc2007数据集上训练，一共20类。准确率为74.5%。
